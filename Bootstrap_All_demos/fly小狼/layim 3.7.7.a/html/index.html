新增配置参数

layim.config(options)的更多可选配置项见下表：

可选项 默认值 类型 用途
isStatus false boolean 是否开启在线状态显示
isCacheSystem false boolean 是否缓存系统消息
isMsgState false boolean 是否显示用户消息状态 error状态必显示,固定相['sending','error']
openWithdraw false boolean 是否开启撤回功能
openMineMsg false boolean 是否开启接受并展示自己对自己发送的消息
openUnReadAwoke false boolean 是否开启未读提醒
msgtypes [] 见下方 array 消息状态数组 基于消息状态,可自由配置[{'name','title','color','background'}]
[{'name':'send','title':'发送中','color':'#DCDCDC','background':'#FBFBEA'},{'name':'reach','title':'送达','color':'#8D8A8A','background':'#DDF3FF'},{'name':'read','title':'已读','color':'#FFF','background':'#00B271'},{'name':'error','title':'失败','color':'#FFF','background':'#B45B3E'}]

新增外部API

*1.setMainShow() //显示主界面
*2.setMainHide() //隐藏主界面
*3.toggleMainDisplay()//显示/隐藏 主界面
*4.imClose() //关闭layim所有界面 并使基本方法失效 必须config 才会恢复
*5.setMsgState(cid,statename,timestamp) //更新消息状态 并同步服务器时间
*6.withdrawChat(cid) //撤回一条信息

新增回调方法

*1. withdraw 撤回消息监听
layim.on('withdraw', function(message){
console.log(message); //获得 message = { username,avatar,id: ,type,content,cid,timestamp,state,mine}
});
*2. delMessage 删除消息监听
layim.on('delMessage', function(message){
console.log(message); //获得 message = { username,avatar,id: ,type,content,cid,timestamp,state,mine}
});
*3. readchat 已读消息回调 系统消息不会回调
layim.on('readchat', function(message){
console.log(message); //获得 message = { username,avatar,id: ,type,content,cid,timestamp,state,mine}
});
*4. groupAdd //新增群成员
layim.on('groupAdd', function(gruopId){
console.log(gruopId); //获得 gruopId
});
*5. groupDel //删除群成员
layim.on('groupDel', function(gruopId){
console.log(gruopId); //获得 gruopId
});
